Detailed Description

template<class TImage, template< class > class TPriority>
class PriorityManual< TImage, TPriority >

This class first returns values that have been specified by a user generated image that contains pixels that indicate "definitely fill first". Once there are no more of these pixels, the superclass's ComputePriority function is called. This is a "template template" class because one of the template parameters (TPriority) depends on another (TImage).
