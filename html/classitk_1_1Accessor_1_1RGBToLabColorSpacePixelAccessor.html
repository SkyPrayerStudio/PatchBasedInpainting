Detailed Description

template<class TInput, class TOutput>
class itk::Accessor::RGBToLabColorSpacePixelAccessor< TInput, TOutput >

Give access to a RGBPixel as if it were in Lab (CIE L*a*b*) Color Space as a Vector type.

This class is intended to be used as parameter of an ImageAdaptor to make an RGBPixel image appear as being an image of Vector pixel type in Lab Color Space.
